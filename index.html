Slack Message Tracker

github.com/khalidsharmarke/SlackMessageTracker | kshar.hopto.org/slack_message_tracker

  • Full Stack Web App that allows users to view historical messages, as a workaround to Personal Slack’s 10k message limitation
  • Event-Based Node.js Microservice hooks to Slack API in order to save new data to Postgres database
  • Leveraged Django ORM to handle storing from microservice and serving data to the front end
  • React UI uses websockets to display real time updates from DB without need for refreshing

A Fake Artist Goes To New York

github.com/khalidsharmarke/A-Fake-Artist-Goes-To-New-York | kshar.hopto.org/fake_artist

  • Web-based implementation of the board game ‘A Fake Artist Goes To New York’ by Oink games
  • Express.JS server handles requests to the index page, room creation, and game state
  • Used socketed connections between server and clients for concurrent game states and data handling
  • Front end implements a modified third-party JS library to give clients drawing capabilities while playing

Reddit Image Scraper

github.com/khalidsharmarke/reddit-image-scraper | kshar.hopto.org/reddit_image_scrapper

  • Chrome Web app that creates a zipped folder of images from a reddit URL for the user to download
  • Express.JS server to handle web requests to itself and to make requests to the Reddit API
  • Front End uses a third party library to download incoming data from the server into a zip folder
  • Able to zip 1,000+ images per request
